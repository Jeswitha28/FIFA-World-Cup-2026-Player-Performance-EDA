# FIFA World Cup 2026 Player Performance Analysis (EDA)

## Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a FIFA World Cup 2026 Player Performance dataset to uncover patterns, trends, and insights related to player and team performance.

Using Python and data visualization techniques, the dataset was analyzed to identify top-performing players, leading goal scorers, assist providers, positional distributions, and relationships between various performance metrics.

---

## Objectives

* Analyze player and team performance statistics.
* Identify top-performing players based on ratings.
* Discover leading goal scorers and assist providers.
* Examine team-wise goal-scoring trends.
* Analyze player distribution across positions.
* Explore correlations among performance indicators.

---

## Dataset Information

The dataset contains detailed player performance statistics, including:

* Assists

## Key Insights

1. Mohamed Elneny achieved the highest player rating.
2. Memphis Zerrouki was the tournament's leading goal scorer.
3. Yassine El Yamiq led all players in assists.
4. Defenders represented the largest positional group.
5. Team scoring patterns varied significantly.
6. Performance Score strongly correlates with Player Rating.
7. Tournament Rating reflects overall player effectiveness.
8. Goal Efficiency provides additional insight into attacking performance.
